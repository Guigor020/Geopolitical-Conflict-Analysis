# Geopolitical-Conflict-Analysis
Interactive Data Dashboard built with Python and Streamlit to analyze fatality trends in the Israeli-Palestinian conflict. Features include geospatial mapping, demographic analysis of victims, and time-series visualization to identify high-violence zones.
